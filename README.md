# CodeAlpha Project: Terminal Hangman (UI Edition)

A modern, cinematic take on the classic text-based Hangman game, built entirely in Python. This project was developed as Task 1 for the Python Programming Internship at CodeAlpha. 

Designed with a moody, noir-inspired aesthetic, the game utilizes terminal block characters and dynamic screen-clearing to create a static, application-like user interface without relying on external graphical libraries. 

## 🚀 Features

*   **Cinematic Terminal UI:** Uses heavy ASCII block characters (`▄`, `█`, `▀`) for a gritty, immersive look.
*   **Static Frame Rendering:** Employs `os.system` calls to clear the console between frames, preventing endless text scrolling.
*   **Input Validation:** Robust error handling to ensure only single alphabetical characters are accepted.
*   **Strict Scope Adherence:** Built strictly to project specifications, utilizing a predefined list of 5 target words and limiting the user to exactly 6 incorrect guesses.

## 🛠️ Technical Stack

*   **Language:** Python
*   **Modules Used:** `random`, `os`
*   **Concepts Demonstrated:** `while` loops, `if-else` control flow, string manipulation, list handling, and clean console I/O.
